# pdf.js-controller

This library provide programming friendly usage of [mozilla/pdf.js](https://github.com/mozilla/pdf.js "mozilla/pdf.js").

- Not official library
- Load pdf and show
- Copy text from pdf
- Click link in pdf

## Installation
